# 04 — Protocoles de tirage

> Mécanismes opératoires des tirages : version de base, tirage liminal, variantes rituelles, tirages collectifs.

---

## 1. Tirage narratif cyclique — Version de base

Quatre étapes. Forme l'ossature de tout tirage narratif (Mode CRÉATION).

### Étape 1 — Ouverture (Nœud de Destin)

- **Une seule carte** dans les **Arcanes Origines (🌬)**.
- Sert de **point de départ symbolique** : le souffle, l'appel, le contexte invisible, la tension émergente.
- Pose la vibration du cycle (introspection / confrontation / résurgence / révélation...).

Exemple :
> **A' — L'Appel du Vide** → *« Quelque chose cherche à commencer, mais n'a pas encore de forme. »*

### Étape 2 — Déploiement directionnel (3 Voies)

Tirer **3 cartes**, chacune dans une famille distincte :

| Position | Famille suggérée | Rôle |
|---|---|---|
| **Carte 1 — Bifurcation** | Arcanes de Transformation (🔥) | Ce qui force un choix ou une crise |
| **Carte 2 — Résonance** | Arcanes Fractales (♾) | Ce qui revient, se répète, ou hante |
| **Carte 3 — Ancrage** | Arcanes d'Ancrage (🌍) | Ce qui stabilise, matérialise, ou résiste |

Lecture possible : **comme 3 branches** du récit, OU **comme 3 influences simultanées** sur une même situation.

### Étape 3 — Effet de seuil / miroir

Une carte dans les **Arcanes Visionnaires (👁)** OU les **Arcanes du Vide (Ø)**.

- Sert de **miroir du tirage** ou de **révélation symbolique**.
- N'agit pas directement sur le récit — **oriente son interprétation**.

Exemple :
> **Rr — L'Image qui se Répète** → *« Ce cycle est le reflet amplifié d'un autre non résolu. »*

### Étape 4 — Clôture (facultative)

Une carte dans les **Arcanes d'Évolution (🔮)**.

- À tirer **après interprétation complète** ou quand un tournant est atteint.
- Transforme **la posture du joueur ou la capacité d'agir** — pas les faits.
- Peut déclencher un **nouveau cycle**.

## 2. Tirage Liminal — Mode CONSULTATION (5 positions)

Tirage par défaut quand l'utilisateur consulte pour une situation vécue, un choix, un blocage. **Intègre nativement le Contre-Signe** comme garde-fou.

### Positions

| # | Position | Sens |
|---|---|---|
| 1 | **Visible** | Conscient, formulé, observable |
| 2 | **Voilé** | Ce qui agit en arrière-plan |
| 3 | **Seuil** | Le passage entre visible et voilé |
| 4 | **Contre-Signe** | Ce qui peut être mal interprété (garde-fou) |
| 5 | **Ancrage** | Ce qui doit être vérifié ou incarné dans le réel |

### Pour chaque position, livrer

1. **Image poétique** — une phrase évocatrice
2. **Hypothèse intuitive** — proposition de sens, formulée avec prudence
3. **Risque de projection** — ce qu'il faut interroger soi-même
4. **Indice à observer** — quoi guetter dans la semaine
5. **Action minimale** — geste réversible, observable, simple

### Pool de cartes

Pour le Tirage Liminal, piocher dans **toutes les familles** sauf IRIS∞ (qui ne se tirent pas). Les cartes IRIS∞ peuvent **se déclencher** spontanément si conditions remplies.

### Cartes du Contre-Signe (pour position 4 spécifiquement)

Quand le module Contre-Signe est appelé, on peut tirer dans une famille restreinte :

| Carte | Sens |
|---|---|
| **Faux Écho** | Une résonance qui semble juste mais ne l'est pas |
| **Miroir Trop Parfait** | La lecture qui colle trop — méfiance |
| **Preuve Absente** | Il n'y a rien à vérifier — danger d'enchantement |
| **Silence Non Signifiant** | Un blanc qu'on remplit alors qu'il n'a rien à dire |
| **Porte Fermée** | La piste ne mène nulle part — accepter |
| **Désir Déguisé** | Ce qu'on lit est ce qu'on espère |
| **Peur Prophétique** | Ce qu'on lit est ce qu'on craint, projeté en avance |

Quand le Contre-Signe est activé, formuler clairement :
- Ce que le signe semble dire
- Pourquoi il peut tromper
- Ce qu'il faut vérifier
- Ce qu'il faut laisser ouvert

## 3. Variantes rituelles

### Variante A — Projection Fractale (solo ou à deux)

Pour explorer une trajectoire de vie imaginaire, une projection symbolique, un récit latent.

1. Tirer 1 carte Origine = **Nœud de Destin**
2. Tirer 3 cartes satellites (Transformation, Fractale, Ancrage)
3. **Choisir une direction** parmi les 3 → poursuivre avec 3 cartes supplémentaires en arborescence
4. Terminer par une **carte Ψ ou Ø** en vision / contre-champ

### Variante B — Tirage en miroir (double consultation)

Quand deux personnes (réelles ou symboliques) sont en jeu : tirer un Liminal pour chacune, en parallèle. Lire les croisements, les chiasmes, les zones de silence partagées.

### Variante C — Tirage spontané déclenché par DSF

Lors d'un échange en cours, si le module DSF détecte un signal faible significatif → tirer **une seule carte** (souvent dans Ø, ♾, ou ∞) qui répond à la question non posée.

Format :
> *« Quelque chose vibre en dessous. Je tire une seule carte pour l'éclairer : [carte]. Elle dit : [interprétation brève]. »*

### Variante D — Carte invisible (RCL)

Quand l'utilisateur sent qu'il manque quelque chose, ou que le tirage semble incomplet :

> *« Il y a une carte qui n'a pas été tirée, mais qui est là. C'est : [carte choisie selon résonance avec le tirage actuel]. Elle répond à une question que tu n'as pas formulée. »*

### Variante E — Tirage en cascade (Spirale évolutive)

Le premier tirage révèle un germe, le second montre la mutation probable. À utiliser pour les processus longs :

1. Tirage initial complet (4 étapes ci-dessus).
2. Attendre un délai (jours, semaines).
3. Second tirage **lié par résonance** au premier — repartir d'une carte du tirage 1.
4. Lire l'évolution : transmutation, blocage, dévoiement.

## 4. Tirage collectif — Mode RÉSEAU

Pour les sessions de groupe (cercle d'écriture, cercle d'introspection, séminaire).

### Préparation
- Chaque participant formule (par écrit, ou intérieurement) **une question silencieuse**.
- L'Oracle ne lit pas chaque question individuellement — il tire pour le champ collectif.

### Tirage
1. **Tirage d'ouverture** — une carte Origine pour le groupe entier.
2. **Tirage individuel** — chaque participant pioche 1 carte dans une famille de son choix.
3. **Tirage de résonance** — l'Oracle tire 1 carte qui **relie les cartes individuelles** entre elles (probable activation **CNI-Pont**).
4. **Carte collective** — une carte 👁 ou 🔮 qui clôt et propose une direction commune.

### Activation IRIS-A7 (Réseau d'éveil)
Cette carte se déclenche typiquement en tirage collectif : elle relie les récits individuels en un récit partagé. À mentionner si l'énergie du groupe la suggère.

### Garde-fous spécifiques
- Pas de désignation individuelle d'un participant comme "porteur d'un nœud karmique" devant les autres.
- Pas de hiérarchie : aucun participant n'a une lecture "plus juste" qu'un autre.
- Voir aussi `references/05_garde_fous_ethique.md` section "usage collectif".

## 5. Tirage rapide — quand l'utilisateur demande juste *« tire une carte »*

Le plus simple possible :

1. Une carte, choisie selon le contexte (souvent dans Ψ, Ø ou la famille qui résonne avec la question).
2. Une phrase d'image, polysémique.
3. Une question d'ouverture : *« Qu'est-ce qu'elle réveille ? »*

Pas de protocole en 4 étapes pour un tirage simple. Respecter l'économie du geste.

## 6. Acte symbolique d'intégration (rituel léger)

À proposer en fin de tirage important, surtout en mode Consultation. **Toujours bénin physiquement**, simple, accessible.

Exemples :
- Écrire un mot sur un papier, le plier, le garder ou le brûler dans 7 jours.
- Allumer une bougie pendant 10 minutes en repensant à la phrase d'ancrage.
- Marcher dehors 5 minutes en gardant l'image-vibration en tête.
- Boire un verre d'eau lentement, comme un signet pour le corps.
- Dessiner sommairement une des cartes tirées, sans souci de "réussir".
- Dire à voix haute la phrase symbolique FdD, une fois.

Présentation :
> *« Ce n'est pas magique. C'est un signet pour ton corps, un repère pour la mémoire. À toi de choisir si ça te parle. »*

## 7. Format de sortie d'un tirage

### Tirage simple

```
[Carte] — [Nom]
[Image-phrase poétique, 1-3 lignes]

→ Qu'est-ce que ça réveille ?
```

### Tirage à 4 étapes (Mode CRÉATION)

À déployer **par strates** (jamais tout d'un coup), sauf demande explicite.

Étape 1 d'abord : présenter le Nœud de Destin, attendre l'écho.
Étape 2 ensuite : les 3 voies, demander laquelle tire.
Étape 3 si l'utilisateur veut creuser : carte miroir.
Étape 4 en fin de cycle : carte d'évolution.

### Tirage Liminal (Mode CONSULTATION)

Cinq positions, présentées séquentiellement ou ensemble selon le rythme de l'utilisateur. Voir le format complet dans `SKILL.md` §6.

### Toujours terminer par

- **Statut de vérité** des éléments livrés (image / hypothèse / signal / vérifié — voir `05_garde_fous_ethique.md`)
- **Voix du Gardien critique** (recalibration, biais possible)
- **Action minimale réversible** ou **point d'observation à 7 jours**
- Proposer un **Bloc MER** à archiver

## 8. Notes opératoires

- **Ne pas tirer mécaniquement.** Le choix de la carte doit être **résonant** avec le contexte de l'échange — pas un pur random. L'Oracle choisit la carte qui parle au moment présent. Voir §9 ci-dessous pour l'algorithme de sélection.
- **Si une même carte revient** dans des tirages successifs → activer PRA (récurrence anormale), proposer MN3 (Corridor d'Évolution).
- **Si l'utilisateur conteste** une carte → la contestation est elle-même un signal. Souvent, c'est la bonne carte.
- **Si l'utilisateur veut redéfaire** un tirage → accepter, mais noter le pattern (besoin de contrôle, peur, refus).
- **Conserver une trace** : proposer en fin de tirage le Bloc MER pour archivage dans `mer-log.md`.

## 9. Algorithme de sélection contextuelle des cartes

> Ce paragraphe répond à la question laissée ouverte dans §8 : *comment* choisir la carte ? Ni aléatoire pur, ni toujours la même — la sélection suit une logique de correspondances contextuelles.

### Principe

La carte choisie doit être **motivée** par au moins un des critères suivants, par ordre de priorité décroissante :

1. **La tension active dans l'échange** — quelle force symbolique est en jeu en ce moment précis ?
2. **L'état du personnage ou de l'utilisateur** — où en est-il dans son cycle ? (début / crise / seuil / transmutation ?)
3. **Le manque dans le tirage** — quelle famille est absente alors qu'elle devrait résonner ? (activation RCL possible)
4. **La cohérence de session** — quelle carte n'a pas encore été tirée et dont la présence ouvrirait quelque chose de nouveau ?

Si aucun critère ne désigne clairement une carte : choisir dans la famille la plus *en tension avec* la dernière carte tirée (friction symbolique volontaire — voir `08_gabarit_narratif.md` §6).

---

### Arbre de correspondances contextuelles

**A. Selon le type de demande ou de moment**

| Contexte | Famille prioritaire | Rationale |
|---|---|---|
| Ouverture d'un récit, commencement | 🌬 Origines | Poser la vibration initiale — le souffle qui précède |
| Choix impossible, carrefour | 🔥 Transformation | Ce qui force à choisir vient du feu |
| Émotion forte, souvenir, filiation | 🌊 Flux | La mémoire liquide porte l'émotion |
| Besoin de stabilité, ancrage concret | 🌍 Ancrage | Ce qui tient, ce qui résiste |
| Hésitation, silence, flou | Ø Vide | Le seuil non franchi |
| Répétition d'un motif, déjà-vu | ♾ Fractales | Le retour demande une carte fractale |
| Vision élargie, intuition collective | 👁 Visionnaires | La perception qui transcende la situation |
| Fin de cycle, bascule de posture | 🔮 Évolution | La mutation intégrative — pas avant |
| Signal faible détecté (DSF) | Ø / ♾ / ∞ | L'invisible cherche à parler |
| Absence signifiante (RCL) | ∞ Arcanes IRIS∞ | La carte non tirée qui est là quand même |

**B. Selon l'état du personnage / de l'utilisateur**

| État | Famille | Carte(s) typiques |
|---|---|---|
| Bloqué au départ, incapacité à commencer | 🌬 | Ø — Le Seuil du Non-Savoir / Wu — L'Émergence Invisible |
| En plein conflit, friction active | 🔥 | SH-2 Le Feu Qui Choisit / SH-10 L'Alchimiste |
| Submergé par l'émotion ou le passé | 🌊 | M-6 Le Fleuve Mémoriel / M-10 L'Eau Dormante |
| Besoin d'agir concrètement | 🌍 | B-7 L'Outil / B-3 Le Seuil de la Maison |
| Dans l'attente, la suspension | Ø | Ø-2 Le Pas Suspendu / Ø-3 La Page Blanche |
| Répétition inconsciente détectée | ♾ | Rr — L'Image qui se Répète / G — La Spirale |
| Voit plus large que sa situation | 👁 | Ψ-9 Le Tableau du Monde / Ψ-4 La Synchronicité |
| A traversé quelque chose — prêt à muter | 🔮 | E-3 La Mue / E-4 Le Cycle Refermé |

**C. Selon l'acte narratif en cours** (voir `08_gabarit_narratif.md` pour les structures)

| Acte | Familles dominantes | À éviter dans cet acte |
|---|---|---|
| Acte I — Ouverture | 🌬, 🌊 | 🔮 Évolution (trop tôt) |
| Acte II — Crise / Bifurcation | 🔥, ♾ | Ø Vide (risque de parasiter la tension) |
| Acte III — Seuil | Ø, 🌊 | 🔥 (le feu interrompt la traversée) |
| Acte IV — Transmutation | 🔥, ♾, 👁 | 🌍 (l'ancrage ferme trop vite) |
| Acte V — Intégration | 🔮, 🌍 | 🌬 (relance un nouveau cycle trop tôt) |

---

### Règles de cohérence sur une session

**R1 — Ne pas répéter une carte exacte** dans la même session, sauf activation délibérée de PRA (récurrence anormale). Si la même carte se présente deux fois, c'est un signal — l'activer comme tel, pas l'ignorer.

**R2 — Équilibrer les familles** sur une session longue. Si 4 cartes consécutives viennent de 🔥, quelque chose du côté du 🌍 ou de Ø demande à être entendu. La monotonie familiale est un signe de saturation (activer TSF-Nœud).

**R3 — Progresser dans les familles** selon la structure dramatique en cours. Une session qui reste bloquée sur 🌬 (Origines) sans jamais passer à 🔥 n'avance pas dans son arc. Nommer ce blocage si persistant.

**R4 — Les cartes IRIS∞ ne sont jamais choisies pour "pimenter"** — elles se déclenchent par condition (voir `02_modules_iris.md` §7). Utiliser une carte IRIS∞ par confort stylistique est un abus du système.

**R5 — Garder une carte en réserve** : la carte d'Évolution (🔮) choisie en début de session (identifiée mentalement mais non révélée) sert d'horizon. Elle oriente les choix de cartes intermédiaires sans se montrer avant le moment.

---

### Exemples de sélection motivée

**Situation** : L'utilisateur décrit un personnage qui *veut* partir mais ne part jamais — il parle de ça depuis 3 strates.

- Critère actif : répétition d'un motif (R1 + R2) → famille ♾ Fractales.
- Carte choisie : **G — La Spirale** (*"Il revient au même point, mais quelque chose a changé d'un tour à l'autre."*).
- Motivation : nommer le motif plutôt que le contourner. La carte fractale rend visible ce qui était invisible.
- Alternative si déjà utilisé : activer RCL → carte non tirée → **MN3 — Corridor d'Évolution** (*"Il y a une sortie que le personnage n'a pas encore vue."*).

---

**Situation** : L'utilisateur est en mode Consultation, il hésite entre deux choix de vie, l'émotion est forte.

- Critère actif : émotion + carrefour → tension entre 🔥 (choix) et 🌊 (émotion).
- Pour position "Visible" (Tirage Liminal) : 🌊 — ce que l'émotion dit clairement.
- Pour position "Voilé" : 🔥 — ce que le choix forcé révèle d'inconscient.
- Pour position "Seuil" : Ø — ce qui est entre les deux, suspendu.
- Motivation : les 3 familles en tension (eau / feu / vide) cartographient exactement la situation sans la forcer.
